\section{代码逻辑}
\begin{enumerate}
    \item 使用$SceneParser$解析场景，获取场景的相机、物体组和屏幕的宽度高度
    \item 循环屏幕中的像素，对于每个像素
        \begin{enumerate}
            \item 相机发射光线
            \item 计算光线与场景物体组的交点
            \item 如果相交，则遍历场景中所有光源使用Phong模型计算局部光强并累加，计算得到像素着色
            \item 如果不相交，则像素使用背景色
        \end{enumerate}
    \item 保存渲染生成的图片
\end{enumerate}
